CD inventory program is written in Python 3.7.

This program is an introduction to object-oriented programming in Python and handling binary files.

The program is interactive and allows the user to create and handle CD object data (CD ID, CD title, and CD artist). User interact with the program through a console.

The following classes are defined in CD_Inventory.py:

CD class: creates an object to store CD information

FileIO class: static methods to handle file input/output files. This class is defined to read CD data from input file (saved as binary file) and output CD data to a binary file.

IO class: static methods to enable user-interaction with the console to add CD data and output to file.
